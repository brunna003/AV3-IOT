Projetos IoT possíveis (1h30) com: DHT, Botão, LED, LDR, Potenciômetro,
Joystick, Buzzer

1.  Estação Climática IoT com Alerta Sonoro

-   Sensores: DHT, LED, buzzer, botão
-   Função: mede temperatura/umidade; alerta sonoro/visual; botão para
    mute; envia dados à nuvem.

2.  Lâmpada Inteligente IoT por Luminosidade

-   Sensores: LDR, LED, botão
-   Função: mede luz; controla LED automaticamente ou via internet;
    botão ativa modo manual.

3.  Controle Remoto IoT com Joystick

-   Sensores: joystick, LED, buzzer
-   Função: envia posição do joystick para a nuvem; dashboard mostra
    X/Y; LEDs e buzzer como feedback.

4.  Cofre IoT com Alarme

-   Sensores: LDR, botão, buzzer, LED
-   Função: luz detectada = cofre aberto; buzzer e LED disparam; IoT
    envia alerta.

5.  Estação de Energia Solar (simulada)

-   Sensores: LDR, potenciômetro, LED
-   Função: LDR simula sol; potenciômetro carga; LED indica geração;
    dashboard IoT exibe gráficos.

6.  Termostato IoT

-   Sensores: DHT, LED, buzzer, potenciômetro
-   Função: mede temperatura; pot ajusta limite; alerta se passar; IoT
    ajusta limite remotamente.

7.  Estação de Jogos IoT com Joystick

-   Sensores: joystick, buzzer, LED
-   Função: joystick envia dados a um mini-game na web; buzzer e LED são
    feedbacks.

8.  Alarme IoT Noturno

-   Sensores: LDR, LED, buzzer, botão
-   Função: arma no escuro; se detectar luz inesperada alarma; IoT
    controla estado.
